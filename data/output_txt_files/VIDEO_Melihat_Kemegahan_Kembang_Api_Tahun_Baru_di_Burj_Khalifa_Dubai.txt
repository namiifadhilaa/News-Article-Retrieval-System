Uni Emirat Arab menyambut Tahun Baru 2024 dengan perayaan kembang api pada Senin (1 1). Ribuan orang berkumpul menyaksikan pertunjukan laser dan kembang api di gedung Burj Khalifa, Dubai. Pertunjukan drone juga diadakan di Ras Al Khaimah, dengan 2.800 drone menyinari langit Dubai. Dubai kerap menjadi salah satu kota yang memiliki pertunjukan kembang api Tahun Baru paling spektakuler. Sebanyak 15.682 kembang api yang dikoordinasikan teknologi komputer berhasil memukau dunia.
