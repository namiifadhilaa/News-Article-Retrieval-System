Kecelakaan ereta api terjadi di Cicalengka, Kabupaten Bandung, Jumat (5 1) pagi. Kecelakaan itu melibatkan dua kereta api. Manajer Humas KAI Daop 2 Bandung Ayep Hanepi mengatakan dua kereta yang terlibat kecelakaan itu adalah KA Turangga dengan KA lokal Bandung Raya. Kecelakaan terjadi di jalur petak Stasiun Cicalengka-Haurpugur pukul 06.53 WIB.    'Kejadian pukul 06.03 WIB di petak Jalur Cicalengka-Haurpugur, KA Turangga dan KA Lokal,' kata Ayep dikutip detikcom. Ayep mengatakan pihaknya belum bisa memastikan terkait jumlah korban dalam insiden kecelakaan ini. Dia menuturkan tim dari KAI sedang menuju ke lokasi. 'Data belum, saya lagi ke lokasi,' ujarnya. Dalam unggahan di akun X @cuancoin21 disebutkan kereta api Turangga bertabrakan dengan dengan KRD. 'INFO TERKINI telah terjadi tabrakan.kereta api Turangga sama KRD di babakan stasion cicalengka jam 6 pagi mohon maaf perjalanan kereta api bandung raya (KRD) terganggu,' demikian tertulis dalam keterangan di unggahan tersebut.
